Создавайте программное обеспечение и работайте вместе с 40 млн других

предложений только для учащихся. Получите бесплатный доступ к отличным
средствам разработки, входящим в пакет GitHub для разработчиков-учащихся.
Станьте техническим лидером в сообществе, получив статус GitHub Campus Expert,
или просто открывайте для себя что-то новое.

##  Visual Studio Code

Пишите код на Python, Java, C++ или любом другом языке, используя Visual
Studio Code. Это компактный, но мощный бесплатный редактор исходного кода с
открытым кодом, работающий независимо от языка. Решение можно использовать на
компьютере с любой ОС — Windows, macOS или Linux. Вы можете писать код,
применяя функцию интеллектуального завершения IntelliSense, и отлаживать
приложение прямо из редактора. Узнайте, как Visual Studio Code помогает
повысить продуктивность. И помните: это совершенно бесплатно.

[ Подробнее ](https://code.visualstudio.com/)
