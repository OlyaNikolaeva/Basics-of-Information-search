##  Вопросы и ответы

Развернуть все  |  Свернуть все

Какие приложения Office я получу в составе Microsoft 365?

Вы получите приложения Office, такие как Word, Excel, PowerPoint, Outlook,
OneNote, OneDrive и т. д.

Подробнее о планах Microsoft 365:

  * [ Для дома ](https://www.microsoft.com/ru-ru/microsoft-365/buy/compare-all-microsoft-365-products)
  * [ Для бизнеса ](https://www.microsoft.com/ru-ru/microsoft-365/business/compare-all-microsoft-365-business-products?&activetab=tab:primaryr2)
  * [ Для крупных предприятий ](https://www.microsoft.com/ru-ru/microsoft-365/compare-microsoft-365-enterprise-plans)

В чем разница между Office 365 и Microsoft 365?

В состав Microsoft 365 входят все приложения и службы Office 365. Набор
Microsoft 365 позволяет пользователям и компаниям добиваться большего с
помощью современных приложений Office, интеллектуальных облачных служб и
первоклассных инструментов защиты данных.

Какие существуют планы Microsoft 365?

[ Планы Microsoft 365 для дома ](https://www.microsoft.com/ru-
ru/microsoft-365/buy/compare-all-microsoft-365-products)

  * Microsoft 365 для семьи 
  * Microsoft 365 персональный 

[ Планы Microsoft 365 для бизнеса ](https://www.microsoft.com/ru-
ru/microsoft-365/business/compare-all-microsoft-365-business-
products?&activetab=tab:primaryr2)

  * Microsoft 365 бизнес базовый 
  * Microsoft 365 бизнес стандарт 
  * Microsoft 365 бизнес премиум 
  * Приложения Microsoft 365 для бизнеса 

[ Планы Microsoft 365 для крупных предприятий ](https://www.microsoft.com/ru-
ru/microsoft-365/compare-microsoft-365-enterprise-plans)

  * Microsoft 365 E3 
  * Microsoft 365 E5 
  * Microsoft 365 F1 
  * Приложения Microsoft 365 для предприятий 

Что произойдет с существующими подписками Office 365 для дома и бизнеса?

Некоторые подписки Office 365 будут автоматически преобразованы в подписки
Microsoft 365. Никаких дополнительных действий для этого не потребуется.

  * Office 365 персональный будет называться Microsoft 365 персональный. 
  * Office 365 для дома станет Microsoft 365 для семьи. 
  * Office 365 бизнес базовый превратится в Microsoft 365 бизнес базовый. 
  * Office 365 бизнес премиум получит название Microsoft 365 бизнес стандарт. 
  * Microsoft 365 бизнес станет называться Microsoft 365 бизнес премиум. 
  * Office 365 бизнес превратится в Приложения Microsoft 365 для бизнеса. 
  * Office 365 профессиональный плюс будет преобразован в Приложения Microsoft 365 для предприятий. 

Следующие планы Office 365 для предприятий не изменятся:

  * Office 365 E1 
  * Office 365 E3 
  * Office 365 E5 

Означает ли это, что Microsoft 365 будет стоить больше?

Нет, стоимость подписки не изменится.

Какие планы Office 365 по-прежнему доступны?

Следующие [ планы Office 365 для предприятий ](https://www.microsoft.com/ru-
ru/microsoft-365/enterprise/compare-office-365-plans) все еще доступны:

  * Office 365 E1 
  * Office 365 E3 
  * Office 365 E5 

Нужен ли доступ к Интернету для работы с Microsoft 365?

Да, доступ к Интернету нужен, чтобы установить и активировать Microsoft 365 и
получить все последние выпуски приложений и служб.

Обратите внимание: если вы уже пользуетесь подпиской, вам не нужно будет
устанавливать приложения заново или покупать другую подписку.

Для работы с приложениями Office, такими как Word, Excel и PowerPoint, доступ
к Интернету не требуется, так как они полностью установлены на компьютере.
